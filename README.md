# ReadNest

A simple blog application where users can read blog posts and admins can add and manage blog posts.  

## Features
- Users can read published blog posts
- Admin panel to add, edit, and delete blog posts

## Tech Stack
- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB

## Usage
1. Clone the repository  
   ```bash
   git clone https://github.com/abirhasan01/ReadNest.git

- Install dependencies
npm install

- Start the application
npm start
